築本後台操作手冊
================


DASHBOARD
---------
各模組數字卡片可點擊，直接跳轉對應頁面。
待處理區塊的每一列也可點擊跳轉。
若有客戶維護合約在 30 天內到期，會出現警示區塊。


PROJECTS
--------
＋ 新增專案：開啟新增表單。
編輯：開啟該筆資料的編輯表單。
刪除：刪除前跳出確認視窗。
分頁按鈕：每頁 10 筆。

表單說明：
- 名稱輸入時 Slug 自動產生，可手動修改。
- 狀態：Draft / Published / Archived，只有 Published 會在官網顯示。
- 排序：數字越小越前面。


ASSETS
------
搜尋框：即時篩選素材名稱。
狀態下拉：篩選 Draft / Published / Archived。
＋ 新增素材：開啟新增表單。
編輯：開啟編輯表單。
刪除：跳出確認視窗，需再次點擊「確認刪除」才執行。
分頁按鈕：每頁 10 筆。

表單說明：
- 標籤：點擊 pill 多選，來源在 Tags 頁面。
- 作者 / 分類 / 關聯專案：下拉選單，來源分別在 Tags 頁面與 Projects 頁面。


API PRODUCTS
------------
搜尋框：即時篩選商品名稱。
狀態下拉：篩選 Draft / Published / Archived。
＋ 新增商品：開啟新增表單。
編輯：開啟編輯表單。
刪除：跳出確認視窗。

表單說明：
- Method：下拉選 GET / POST / PUT / DELETE。
- Request / Response 範例：填入 JSON 格式。
- 內部備注：只在後台顯示，不對外公開。
- 標籤：點擊 pill 多選。


IDEAS
-----
搜尋框：即時篩選點子標題。
狀態下拉：篩選 新點子 / 討論中 / 已採用 / 已放棄。
＋ 新增點子：開啟新增表單。
狀態 badge：直接點擊循環切換狀態。
編輯：開啟編輯表單。
刪除：跳出確認視窗。


MEETINGS
--------
搜尋框：即時篩選會議標題。
狀態下拉：篩選 Scheduled / Done / Cancelled。
＋ 新增會議：開啟新增表單。
編輯：開啟編輯表單。
刪除：跳出確認視窗。

表單內待辦清單：
- 新增：輸入內容、指派負責人、設定截止日後點擊新增。
- 狀態 badge：點擊循環切換 未開始 → 進行中 → 已完成。
- ✕：移除該筆待辦，不跳確認直接執行。


REQUESTS
--------
搜尋框：即時篩選詢問者或主旨。
點擊整列：開啟詳細內容 Modal。
狀態 badge：在表格直接點擊循環切換 新進 → 已讀 → 已回覆。

Modal 內：
- 狀態切換按鈕：同上。
- 儲存備註：填入內部說明後點擊儲存，不對外顯示。

注意：此頁無刪除功能，詢問紀錄應保留。


CLIENTS
-------
＋ 新增客戶：開啟新增表單。
編輯：開啟編輯表單。
刪除：跳出確認視窗。

表單說明：
- 合作狀態：Active / Maintaining / Ended。
- 維護到期日：30 天內到期會在 Dashboard 出現警示。
- 負責成員：下拉選單，來源在 Members 頁面。


MEMBERS
-------
＋ 新增成員：開啟新增表單，填入姓名與職稱。
編輯：修改姓名或職稱。
刪除：跳出確認視窗。


TAGS / 分類 / 作者
-----------------
頁籤切換：點擊 Tags / 分類 / 作者 切換清單，三個操作方式相同。
編輯：欄位變成可輸入狀態，同時出現儲存按鈕。
儲存：確認修改。
刪除：直接刪除，不跳確認視窗。
＋ 新增：在清單下方輸入名稱後點擊新增。


操作日誌
--------
關鍵字搜尋：篩選操作者名稱或對象名稱。
動作下拉：篩選 新增 / 編輯 / 刪除 / 狀態變更。
模組下拉：篩選特定模組。
日期範圍：設定起始與結束日期。
重置：清除所有篩選條件。
備注：開啟備注視窗，可附加說明，不修改原始日誌內容。
分頁按鈕：每頁 20 筆。

注意：日誌由後端自動寫入，無法手動新增或刪除。


帳號管理
--------
儲存變更：修改自己的使用者名稱或 Email。
更新密碼：需輸入舊密碼，新密碼至少 8 個字元。
編輯（帳號列表）：修改其他帳號資料或重設密碼，密碼留空則不更改。
停用（帳號列表）：停用該帳號，對方將無法登入，自己的帳號不會出現此按鈕。
新增帳號：填入名稱、Email、密碼後新增，立即出現在列表。

注意：停用帳號後無法從前端復原，需直接操作資料庫還原。
